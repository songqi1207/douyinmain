# 图片类工作流盘点与我们的一期目录

数据来源：2026-07-18 对会员站工作流列表的只读检查。对方列表使用多标签分类；“图片”不是单一类别，而是以下四种产品能力：

1. **纯生图**：关键词/提示词批量出图。
2. **图文卡片**：小红书笔记、菜谱、海报、语录等“文案 + 图片”。
3. **商品/人物图片**：详情页、换装、抠图、换背景、换脸。
4. **图片转视频**：数字人口播、动作、换人、书单轮播和动态壁纸。

## 我们的 FastAPI 前端原型

- 启动：`uvicorn fastapi_app:app --host 0.0.0.0 --port 8000`
- 页面：`/workflows`
- 分类接口：`GET /api/workflow-categories`
- 搜索接口：`GET /api/workflows?q=养生&kind=图文卡片`
- 详情接口：`GET /api/workflows/G218`
- 统一生成入口（暂未接 provider）：`POST /api/workflows/G218/run`

前端参考了对方站点的卡片式暗色布局，但目录和数据是我们自己的；第三方密钥只允许留在服务端，浏览器只提交用户输入和工作流编号。
